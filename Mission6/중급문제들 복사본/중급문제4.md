-- 4. `orders` 테이블에서 각 날짜별 총 주문 건수를 `order_count` 라는 이름으로 계산하고, 하루 총 주문 건수가 80건 이상인 날짜만 조회한 뒤, 주문 건수가 많은 순서대로 정렬하세요.


orders` 테이블에서 각 날짜별 총 주문 건수를 `order_count` 라는 이름으로 계산하고

    COUNT(id)  AS order_count
    GROUP BY date 


하루 총 주문 건수가 80건 이상인 날짜만 조회한 뒤,
    HAVING  order_count >= 80

주문 건수가 많은 순서대로 정렬하세요.

    SELECT date o
      FROM orders ORDER BY order_count DESC

izza_place=# SELECT date, COUNT(id) AS order_count
FROM orders
GROUP BY date
HAVING COUNT(id) >= 80
ORDER BY order_count DESC;
